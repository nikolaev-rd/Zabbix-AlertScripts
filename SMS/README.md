 * [send-sms-mfms-post.py](send-sms-mfms-post.py)
 Send SMS via SMS Gateway (MFM Solutions), method ThreeDSecure (HTTP POST).
 You need to specify: 
   - SMS_GATEWAY_HOST
   - SMS_GATEWAY_PORT (default: `30080`)
